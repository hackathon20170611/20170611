# 試す
Demo : https://dononiku-meetapp.herokuapp.com/

1. 上記ウェブページにアクセスします
2. 右上メニューから「認証」をクリック
3. ユーザー名＆パスワードを入力します
   テスト用ユーザー：admin, パスワード：admin
4. 右上「エンティティ」＞「Meet Judge」
5. Meed Judge を追加
6. 肉画像をアップロード！
7. スコアと評価が出ます。


# how to deploy
Prerequirements
- Your local [JHipster](https://jhipster.github.io/) development environment
- Heroku registration

1. clone this repository.
2. ```heroku login```
2. run ```jhipster heroku```
3. input app information
4. ```heroku open```
